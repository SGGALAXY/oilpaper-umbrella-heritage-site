# AI 协同开发记录

## 项目目标

围绕“伞制作技艺（油纸伞制作技艺）”制作一个非遗主题网站，满足 Web 前端开发技术课程要求：Bootstrap、响应式布局、JavaScript 交互、GitHub Pages 可发布、README 文档与素材来源说明。

## 使用工具

- 需求拆解：Codex
- UI 方向：web-design-engineer skill
- 图片生成：imagegen fallback CLI / GPT Image 2
- 代码开发：HTML、CSS、JavaScript、Bootstrap 5
- 调试验证：本地浏览器预览、静态文件检查

## 协同流程

1. 根据任务截图提取硬性要求：非遗主题、至少 3 个导航栏目、Bootstrap 栅格、至少 3 种 Bootstrap 组件、至少 2 个 JS 交互、代码分离、README。
2. 确定主题为“泸州分水油纸伞数字展馆”，视觉方向为“东方非遗 × 高级展陈 × 宣纸留白”。
3. 使用 Web 检索核对项目事实，包括 2008 年列入国家级非遗名录、项目编号 Ⅷ-140、传统材料与工序信息。
4. 第一版使用 GPT Image 2 生成主视觉、工坊、伞面细节、工序长卷。
5. 根据反馈升级为“高级档案馆”方向，减少方块卡片和模板化轮播。
6. 使用 GPT Image 2 重新生成 6 张写实工序图：选竹、制骨、穿线、裱面、绘花、上油。
7. 建立新版页面结构：档案封面、档案索引、材料笔记、六个工序标本、影像墙、传承笔记、访客簿。
8. 编写 JavaScript：滚动进度、滚动浮现、工序索引定位、当前工序高亮、图片模态框、留言动态渲染、知识问答。
9. 压缩图片为 WebP，保证页面加载速度。

## 人工审核与修改建议

提交前建议团队成员按以下清单复核：

- 将 README 中的团队成员与 GitHub Pages 链接替换为真实信息。
- 如课程要求必须提交“界面效果图”，可打开网页后截图首屏、工序、展廊、留言四个区域。
- 如果老师要求不用 CDN，可下载 Bootstrap 到本地并修改 `index.html` 引用路径。

## 素材合规说明

网页未直接搬运网络图片。主要视觉图与 6 张工序图均为 AI 生成素材，生成提示词与输出文件见 `docs/image-prompts.md`。事实信息来源见 `README.md` 和网页页脚。
